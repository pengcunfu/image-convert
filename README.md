# 图片格式转换工具

一个简单易用的图片格式转换工具，支持将多种图片格式批量转换为 JPG 格式。

## 功能特点

- ✅ 支持多种图片格式：JPG, PNG, BMP, GIF, TIFF, WEBP, JFIF, AVIF, HEIC, HEIF, SVG, ICO 等
- ✅ 批量转换，自动处理整个文件夹
- ✅ 自动处理透明背景（转换为白色背景）
- ✅ 图形界面，操作简单直观
- ✅ 实时显示转换日志和进度
- ✅ 多种转换引擎（OpenCV, imageio）确保兼容性

## 使用方法

### 方式一：使用可执行文件（推荐）

1. 双击运行 `图片格式转换工具.exe`
2. 点击"浏览"按钮选择源文件夹（包含要转换的图片）
3. 点击"浏览"按钮选择目标文件夹（保存转换后的图片）
4. 点击"开始转换"按钮
5. 等待转换完成，查看日志了解详细信息

### 方式二：使用 Python 脚本

#### 图形界面版本

```bash
python convert_images_gui.py
```

#### 命令行版本

```bash
python convert_images.py
```

注意：命令行版本默认从 `images2` 文件夹读取图片，转换后保存到 `images` 文件夹。

## 安装依赖

如果要运行 Python 脚本，需要先安装依赖：

```bash
pip install -r requirements.txt
```

## 打包为可执行文件

运行打包脚本：

```bash
build.bat
```

打包完成后，可执行文件位于 `dist` 文件夹中。

## 技术栈

- **Python 3.x**
- **OpenCV** - 图片处理
- **imageio** - 图片读写
- **NumPy** - 数值计算
- **tkinter** - 图形界面
- **PyInstaller** - 打包工具

## 注意事项

- 转换过程中会自动覆盖目标文件夹中同名的文件
- 透明背景的图片会被转换为白色背景
- 转换质量设置为 95%，确保高质量输出
- 如果某个图片转换失败，程序会尝试其他转换引擎
